# 功能介绍
* 1) 接收显示框，默认一个比较淡的护眼色，现在还没确定
* 2）发送数据框。
* 3）串口设置
* 4）状态栏、串口连接、发送、接收数据显示
* 5）日志dump到文件
* 6）定时发送
* 7）清空发送、接收框
* 8）时间戳显示
