# 功能介绍

* 接收显示框，默认一个比较淡的护眼色，现在还没确定
* 发送数据框。
* 串口设置
* 状态栏、串口连接、发送、接收数据显示
* 日志dump到文件
* 定时发送
* 清空发送、接收框
* 时间戳显示
